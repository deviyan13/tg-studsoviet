\# Telegram Quest Form Bot



Бот для быстрого создания формы регистрации на мероприятие прямо из Telegram.



Проект помогает автоматизировать рутинную подготовку регистрации: вместо ручного создания формы, оформления шапки и связки с таблицей бот сам генерирует картинку, создает нужные данные и отправляет все в Google-связку.



\## Что умеет бот



\- создает форму регистрации через диалог в Telegram;

\- принимает название мероприятия;

\- принимает максимальное и минимальное количество участников;

\- генерирует обложку для формы в PNG;

\- отправляет картинку и отдельным файлом для скачивания;

\- передает данные в Google Apps Script для создания формы и таблицы;

\- поддерживает работу только для разрешенных пользователей;

\- имеет простые админ-команды для просмотра доступа.



\## Используемые технологии



\- \*\*Python\*\*

\- \*\*python-telegram-bot\*\*

\- \*\*requests\*\*

\- \*\*asyncio\*\*

\- \*\*dotenv\*\*

\- \*\*Google Apps Script\*\*

\- \*\*Git\*\*



\## Особенности



\- ограничение доступа по списку разрешенных пользователей;

\- генерация изображения с адаптацией текста под длину названия;

\- отправка результата сразу в удобном виде: фото + файл.



\## Переменные окружения



Для запуска нужны:



\- `TELEGRAM\_BOT\_TOKEN` — токен Telegram-бота;

\- `ALLOWED\_USERS` — список разрешенных user\_id через запятую;

\- `ADMIN\_USER\_ID` — ID администратора;

\- `GOOGLE\_SCRIPT\_URL` — URL Google Apps Script;

\- `GOOGLE\_DRIVE\_FOLDER\_ID` — ID папки в Google Drive;

\- `WEBHOOK\_URL` — URL для webhook-режима;

\- `PORT` — порт для запуска webhook.
